# Market_basket_analysis
Objective: to identify cross-selling, i.e. products that sell well together.  
We work with anonymous data provided by an e-shop.  

### Folder hierarchy
```
Market Basket Analysis
|   basket_analysis.ipynb : the principal notebook with the analysis and a little result
|   
+---data
|       Data test.xlsx : The file provided by the e-shop with the original data
|       frequent_itemset.csv : the list of products with their frequencies.
|       
+---utils
|       basket_analysis_utils.py : Some functions of use.
```

### Data test.xlsx 
The available fields in the file:
- id_customer	
- id_order	
- product_id	
- product_quantity
